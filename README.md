# Google-search-Trend-Analysis

## Project Overview
This project analyzes Google Search Trends data to identify keyword popularity, search patterns, and market trends. The analysis helps understand user behavior and emerging trends using data analytics techniques.

## Objectives
- Analyze search trends over time
- Identify top trending keywords
- Compare popularity of different search terms
- Visualize trends using charts

## Tools & Technologies Used
- Python
- pytrends
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Steps Performed
- Data Collection from Google Trends using pytrends
- Data Visualization

## Key Insights
- Identified high-growth search terms
- Analyzed seasonal trends
- Compared keyword performance globally
